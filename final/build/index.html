Triggers

  • Hover on/off circular blue shopping bag icon
  • Hover on/off image container
  • Hover on/off left/right arrows
  • Click on left/right arrows
  • Hover on/off inactive heart icon
  • Click on heart icon
  • Hover on/off active heart icon

Rules

Hover on shopping bag icon

  • Shopping bag icon scales 5% larger at 0.2s and turns a darker shade of blue
  • Underline underneath product name fades in at 0.2s
  • Gray tooltip that says "Add to Bag" pops out and expands from above the shopping bag icon

Hover off shopping bag icon

  • Shopping bag icon shrinks to original size and turns back to original, lighter blue color
  • Underline underneath product name fades out at 0.2s
  • Gray tooltip that says "Add to Bag" shrinks back above the shopping bag icon

Hover on image container

  • Black overlay at 20% opacity fades over image container
  • Left and right arrow fade in on respective sides
  • Carousel indicator fades in at the bottom center of the image container

Hover off image container

  • 20% opacity black overlay, left and right arrow, and carousel indicator fades out

Hover on left/right arrows

  • Cursor turns to pointer

Hover off left/right arrows

  • Pointer turns back to cursor

Click on left/right arrows

  • If right arrow clicked, current image slides to the left while next image in carousel slides in from the right
  • If left arrow clicked, current image slides to the right whle next image in carousel slides in from the left
  • Carousel indicator fills the circle respective to which image is being shown in the carousel
  • At the start and end of the carousel, clicking on the left and right arrow will cycle the carousel to the last and first picture respectively

Hover on inactive heart icon

  • Cursor turns to pointer
  • Heart icon scales 5% larger at 0.2s
  • Gray tooltip that says "Save" pops out and expands above heart icon

Hover off inactive heart icon

  • Pointer turns back to cursor
  • Heart icon shrinks back to original size
  • Gray tooltip that says "Save" shrinks back above heart icon

Click on heart icon

  • Database is updated for the user's saved products
  • When inactive: heart icon fades to a red color and pulses once before returning to original size, gray tooltip text changes from "Save" to "Unsave"
  • When active: heart icon's red color fades away to return back to gray outline while pulsing once simultaneously, gray tooltip text changes from "Unsave" to "Save"

Hover on active heart icon

  • Cursor turns to pointer
  • Heart icon scales 5% larger at 0.2s
  • Gray tooltip that says "Unsave" pops out and expands above heart icon

Hover off active heart icon

  • Pointer turns back to cursor
  • Heart icon shrinks back to original size
  • Gray tooltip that says "Unsave" shrinks back above heart icon
