IDM241 Alpha Description

Microinteraction: Ikea 'Add to Bag' Button - Ikea Website

Triggers

  • Hover on/off circular blue shopping bag icon

Rules

Hover over shopping bag icon

  • Cursor turns to pointer
  • Button will turn a darker shade of blue
  • Name of the product will become underlined immediately

Hover off shopping bag icon

  • Pointer turns back to cursor
  • Color of button turns back to original, lighter blue
  • Underline under the name of product disappears immediately
